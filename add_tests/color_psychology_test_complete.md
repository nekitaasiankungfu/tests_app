---

## ОПИСАНИЕ ПРОЕКТА

Это полная спецификация оригинального теста цветовой психологии для диагностики текущего эмоционального состояния, стресса, потребностей и психологических установок через выбор и ранжирование цветов. Тест использует научно доказанную связь между цветовыми предпочтениями и психологическим состоянием человека. Тест предназначен для мобильного приложения на Android (Kotlin + Jetpack Compose).

**ВАЖНО: Это оригинальный тест, разработанный на основе общих принципов цветовой психологии, но отличающийся от классического теста Люшера методологией, набором цветов, интерпретацией и системой подсчета. Авторские права не нарушаются.**

**Что включено:**
- Полная методология и научная база
- 3 этапа тестирования (выбор, ранжирование, ассоциации)
- 10 цветов с психологическими значениями
- 8 психологических шкал
- Анализ текущего состояния и потребностей
- Детальная интерпретация результатов
- Рекомендации по управлению состоянием
- Отслеживание динамики состояния

## ИНСТРУКЦИИ ДЛЯ CLAUDE CODE

**При создании приложения:**

1. **Data модели** - Color, TestStage, ColorSelection, ColorRanking, ColorAssociation, PsychologicalScales, Pattern
2. **Color Rendering** - КРИТИЧЕСКИ ВАЖНО точное отображение цветов (hex коды)
3. **Test Flow**:
   - Этап 1: Быстрый выбор (30 сек)
   - Этап 2: Полное ранжирование (60 сек)
   - Этап 3: Ассоциации (без лимита)
4. **Scoring Engine**:
   - Анализ позиций цветов
   - Расчёт 8 психологических шкал
   - Выявление паттернов (burnout, depression, anxiety)
   - Анализ ассоциаций
5. **UI особенности**:
   - Большие, яркие цветовые квадраты
   - Drag & drop для ранжирования
   - Таймер для этапов 1-2
   - Минималистичный дизайн
6. **Результаты**:
   - 8 шкал с интерпретацией
   - Выявленные паттерны
   - Текущее состояние
   - Рекомендации
   - **КРИЗИСНЫЕ РЕСУРСЫ** при тревожных паттернах
7. **Tracking**: История результатов для отслеживания динамики

**Важно:**
- НЕ диагностический инструмент
- Измеряет ТЕКУЩЕЕ состояние (не личность)
- Обязательные trigger warnings и кризисные ресурсы
- Рекомендация профессиональной помощи при тревожных результатах
- Точность цветопередачи критична
