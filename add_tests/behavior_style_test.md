## ОПИСАНИЕ ПРОЕКТА

Это полная спецификация психологического теста DISC для определения поведенческого профиля личности. Тест основан на модели Уильяма Марстона и измеряет четыре ключевых фактора поведения: Доминирование (Dominance), Влияние (Influence), Стабильность (Steadiness) и Сознательность (Conscientiousness). Тест предназначен для мобильного приложения на Android (Kotlin + Jetpack Compose).

**Что включено:**
- Полная методология и научная база
- 56 вопросов (14 на каждый фактор)
- 4 основных шкалы DISC
- Система подсчета баллов и нормализации
- Детальная интерпретация с 16 классическими профилями DISC
- Применение в карьере, коммуникации и лидерстве
- Рекомендации для развития каждого типа

## ИНСТРУКЦИИ ДЛЯ CLAUDE CODE

**При создании приложения:**

1. **Data модели** - Question, Scale, Profile, Result
2. **JSON файлы** - все 56 вопросов (14 на каждый фактор)
3. **TestScorer**:
   - Нет реверсивных вопросов
   - Нормализация: `((raw - 1) / 4) * 100` для каждого фактора
   - Определение профиля: топ-2 фактора с баллами >60
4. **UI особенности**:
   - **4 цвета DISC**: D=красный, I=жёлтый, S=зелёный, C=синий
   - Круговая диаграмма для визуализации 4 факторов
   - Экран результатов с профилем + описание
5. **16 профилей** - определение на основе комбинации топ-2 факторов
6. **Практическое применение** - секции о карьере, коммуникации, лидерстве

**Важно:**
- Профессиональный, бизнес-ориентированный тон
- Фокус на применении в работе и команде
- Подчёркивать, что нет "плохих" профилей
- Акцент на развитии и балансе
