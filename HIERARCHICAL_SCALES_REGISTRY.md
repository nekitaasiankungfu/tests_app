- New scales can be added incrementally as needed
- User confirmed: scales can be "on reserve" (empty initially) - they will be filled as new tests are added

## Scale ID Naming Conventions

- Use snake_case for all scale IDs
- Prefix categories with `cat_` to distinguish from scales
- For scales that could conflict (e.g., `impulsiveness` vs `impulsivity_dark`), add clarifying suffix
- Keep IDs concise but descriptive
